# Hidden Markov Models

## Directory Setup

All training and validation data is placed in the root of this repository after cloning. The file structure should be as follows:

```
data
   - CN
   - EN
   - FR
   - SG
```

## Evaluation script
```
./eval.sh [dataset] [qn]
# example
./eval.sh EN 2
./eval.sh SG 3
```
